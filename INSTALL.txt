Install steps
1. Open your project root.
2. Replace these 3 files with the versions from this package:
   - src/app/features/cycles/cycle-one/cycle-one.ts
   - src/app/features/cycles/cycle-one/cycle-one.html
   - src/app/features/cycles/cycle-one/cycle-one.css
3. Run your normal build/start command.

What this package changes
- Opponent-side matchup styling now uses that owner's selected favorite team theme.
- Mobile matchup score cards also reflect each team's favorite theme.
- The Cycle Transparency section now uses a dark blue theme that matches the rest of the page.
- Mobile overflow/centering on the player-card area is tightened so the matchup content no longer hangs off the right side as easily.
