How Fantasy Cycles Work

Every player gets six NHL team games.

A fantasy cycle is not a calendar week. Each player scores from the next six games played by that player's NHL team, so every roster spot can move at its own pace.

Show the full cycle explanation See why players finish at different times and how the next window begins. ⌄
@if (isCommissioner() && developerToolsEnabled) { Run 7-vs-4 Simulator }
1
Each roster spot follows six games

The six numbered circles on a player card represent six scheduled games for that player's NHL team.

2
Players finish at different times

NHL teams have different schedules. One player may finish all six games before another player has reached Game 4.

3
All six games still count

Your matchup score keeps adding each player's fantasy points until every active roster spot has completed its own six-game window.

4
The next window can begin early

When a player finishes Game 6, that player's next NHL team game begins the next cycle—even while other players are still finishing the current one.

Simple example

If Player A has finished six games and Player B has only finished four, Player A's next NHL game begins Player A's next cycle. Player B's final two games still count toward the current matchup. No game is skipped, and no game counts twice.

How to read the circles on each player card

Read the circles from 1 to 6. Each circle shows what happened with one scheduled NHL team game in that player's current fantasy window.
