Expected mobile flow
--------------------
1. Open Add / Drop Player.
2. Select a free agent or waiver player.
3. The long player pool is replaced by a dedicated roster-slot screen.
4. Review the incoming player's current six-game block and earliest fair cycle.
5. Select a compatible same-position roster slot.
6. Review that slot's current locked window and exact activation cycle.
7. Confirm from the fixed bottom action bar.

Fairness example
----------------
- Current roster player: Cycle 5.
- Incoming free agent: already Cycle 6, Game 2.
- Result: the incoming player is reserved but cannot activate before Cycle 7.
- If the old player remains active, that player keeps the Cycle 6 slot window.
- If the old player was moved to IR, the active slot may stay empty for Cycle 6
  rather than receiving the incoming player's already-played Cycle 6 games.
